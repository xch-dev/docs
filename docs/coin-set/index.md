# Introduction

Before you start using the Wallet SDK, it's a good idea to have a basic understanding of the Coin Set Model that the Chia blockchain uses.

:::info
The official docs for the Coin Set Model is available on the [Chia docs](https://docs.chia.net/chia-blockchain/coin-set-model/intro/). However, this website focuses on learning it from the perspective of Chia development using the Wallet SDK.
:::
